# Control-de-accesos-con-RFID
Sistema que permite controlar la entrada de personas usando tarjetas RFID, mejorando la seguridad de un espacio.
#abstract ESPAÑOL
El proyecto desarrolla un sistema de control de accesos mediante tecnología RFID, gestionado por un microcontrolador ESP32. El sistema identifica a los usuarios mediante tarjetas RFID, consulta una base de datos MySQL a través de WiFi y decide si el acceso es autorizado. Además, muestra mensajes en una pantalla LCD, acciona un relé o servomotor y registra todos los accesos con fecha, hora y resultado de cada intento realizado.
#abstract ÍNGLES
The project develops an RFID-based access control system managed by an ESP32 microcontroller. The system identifies users through RFID cards, queries a MySQL database via WiFi, and decides whether access is authorized. It also displays messages on an LCD screen, activates a relay or servomotor, and logs all access attempts with date, time, and the result of each attempt. The design focuses on security, reliability, and practical real-world application.
